# Quantum Optimization — Interactive Web Presentation

An advanced, visually immersive web-based presentation explaining how quantum computing is revolutionizing optimization problems.

🌐 **Live Demo:** https://quantum-optimization.netlify.app/

---

## 🚀 Features

- 🎨 Modern glassmorphism UI with gradient aesthetics
- ⚡ Smooth animations using GSAP & ScrollTrigger
- 📊 Structured academic content (OT subject ready)
- 🧠 Covers deep concepts:
  - NP-Hard problems
  - Quantum superposition & entanglement
  - QAOA, Grover’s Algorithm, Quantum Annealing
- 📱 Fully responsive design
- 🎯 Scroll-based navigation with progress tracking

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- GSAP (Animations)
